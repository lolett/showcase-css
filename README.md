# Modern CSS Showcase (DIW)

## Overview
Static HTML/CSS project for **DIW** to showcase modern CSS: pseudoclasses, transitions/animations, blend effects, and responsive design. :contentReference[oaicite:0]{index=0}

## Live Demo
```bash
https://lolett.github.io/showcase-css/
```

## Requirements Covered
- Semantic HTML (`header/main/section/footer`)
- ≥4 pseudoclasses
- `@keyframes` + `animation` + `transition`
- Images + `mix-blend-mode` + background properties
- Google Fonts + Font Awesome
- Responsive design (media queries + relative units)
